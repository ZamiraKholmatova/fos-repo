# Рекомендации преподавателям по использованию ресурсов

## Принцип подбора

Ресурс включается в дисциплину, если он обеспечивает конкретный индикатор и используется хотя бы в одном КИМ. Для каждого ресурса указываются: назначение, связанный КИМ, способ доступа и условия использования (см. [карточку ресурса](../../resources/README.md)).

## Соответствие ресурсов индикаторам

| Индикаторы | Основные ресурсы |
|---|---|
| LC-1.1, LC-1.3 | Литература по управлению проектами и бизнес-анализу; ГОСТ Р 71539–2024, ГОСТ Р 59277–2020; шаблоны устава проекта и ТЗ |
| LC-4.1, LC-4.2 | PMBOK; описания CRISP-DM и CRISP-ML(Q); инструментальные средства (GanttProject, Microsoft Project, Miro, Asana, Kaiten) |
| LC-7.1, LC-7.2, LC-7.3 | Публикации по экономике ИИ; методики расчета совокупной стоимости владения и окупаемости; материалы по рискам систем ИИ |
| LC-6.1 … LC-6.4 | Материалы по цифровой трансформации и построению цифровой экосистемы организации |

## Требования к обновлению

- проверяйте работоспособность ссылок и актуальность изданий перед началом семестра;
- версии программного обеспечения фиксируйте в описании КИМ — расхождение версий частая причина затруднений на занятии;
- при замене ресурса убедитесь, что покрытие индикаторов сохранилось.

## Данные для практических работ

Приоритет отдается открытым наборам данных с ясной лицензией. Данные индустриальных партнеров используются только при наличии согласия правообладателя, в разрешенном объеме и без размещения в публичном репозитории. Для каждого учебного набора данных фиксируйте: источник, лицензию, наличие персональных данных и порядок обезличивания.

## Использование генеративных моделей в преподавании

Генеративные модели могут применяться для подготовки вариантов заданий, примеров кейсов и черновиков методических материалов. Любой сгенерированный материал перед выдачей обучающимся проверяется преподавателем на фактическую точность — особенно ссылки на стандарты, нормативные акты и числовые данные. Промпты, показавшие устойчивый результат, сохраняйте в разделе [llm-prompts](../../resources/llm-prompts/README.md).
