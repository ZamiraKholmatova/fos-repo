# Информационные ресурсы дисциплины

Раздел содержит ресурсы, обеспечивающие выполнение контрольно-измерительных материалов. Ресурс включается в фонд, если он обеспечивает конкретный индикатор и используется хотя бы в одном КИМ.

| Раздел | Содержание | Связанные КИМ |
|---|---|---|
| [Учебники и пособия](textbooks/README.md) | Литература по управлению проектами и машинному обучению | КИМ-1.1, КИМ-1.2, КИМ-2.1, КИМ-2.2 |
| [Статьи и стандарты](papers/README.md) | ГОСТ Р по системам ИИ, CRISP-DM, CRISP-ML(Q), публикации по экономике ИИ | КИМ-1.2, КИМ-2.1, КИМ-2.4, Проект |
| [Наборы данных](datasets/README.md) | Синтетические учебные наборы с заложенными дефектами качества | КИМ-1.1, КИМ-2.2, Проект |
| [Программное обеспечение](software/python-libs/README.md) | Инструментальные средства управления проектами и расчетов | КИМ-2.1, КИМ-2.2, КИМ-2.3 |
| [Банк задач](problem-banks/README.md) | Практические и кейсовые задания | все практические КИМ |
| [Банк тестовых заданий](test-banks/README.md) | Материалы рубежного контроля | КИМ-2.4 |
| [Промпты для генеративных моделей](llm-prompts/README.md) | Проверенные шаблоны запросов и ограничения их применения | все КИМ |
| [Сравнение инструментов](benchmarks/README.md) | Критерии выбора средств управления проектами | КИМ-2.1, КИМ-2.3 |
| [Прочие ресурсы](other/README.md) | Шаблоны документов и справочные материалы | все КИМ |

## Программные инструменты

Отдельно от информационных ресурсов репозиторий содержит [рабочие скрипты](../tools/README.md): валидатор репозитория, калькулятор экономики ИИ-проекта, генератор вариантов контрольной работы и генератор учебных наборов данных. Все скрипты работают на Python 3.10+ без сторонних зависимостей.

## Требования к обновлению

Перед началом семестра проверяется работоспособность ссылок, актуальность изданий и версий программного обеспечения. При замене ресурса проверяется, что покрытие индикаторов сохранилось: для этого удобно запустить `python tools/validate_repository.py`.
