# Бенчмарки

В настоящей дисциплине бенчмарки моделей машинного обучения не используются: дисциплина посвящена организации и управлению ИИ-проектом, а не разработке и сравнению моделей.

Раздел может использоваться для сопоставления **инструментальных средств управления проектами** по критериям, применяемым в КИМ-2.1.

| Объект сравнения | Критерии | Связанный КИМ |
|---|---|---|
| Средства календарного планирования (GanttProject, Microsoft Project, Miro) | Поддержка зависимостей, критического пути, экспорта, совместной работы | КИМ-2.1 |
| Системы управления задачами (Asana, Kaiten) | Ведение статусов, отчетность, распределение ответственности | КИМ-2.3 |

Сравнение моделей качества систем ИИ рассматривается в дисциплине «Организация процесса разработки ML-систем».
