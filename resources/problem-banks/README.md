| Формализация бизнес-проблемы по описанию предметной области | Отработка перехода от описания «как есть» к постановке задачи ИИ | КИМ-1.1 | LC-1.1 |

| Расчет окупаемости ИИ-решения при заданных параметрах | Отработка технико-экономической оценки | КИМ-2.2, КИМ-2.4 | LC-7.1 |
