# Банк тестовых заданий

Материалы для рубежного контроля (КИМ-2.4) и самопроверки.

| Раздел | Содержание | Индикаторы | Количество заданий |
|---|---|---|---|
| Жизненный цикл систем ИИ | Этапы, их содержание и последовательность | LC-4.1 | *(заполняется преподавателем)* |
| Методологии организации разработки | CRISP-DM, CRISP-ML(Q), гибкие и гибридные подходы | LC-4.1 | |
| Планирование проекта | ИСР, календарный план, критический путь, ресурсы | LC-4.1 | |
| Технико-экономические показатели | Показатели эффективности, окупаемость, стоимость владения | LC-7.1 | |
| Риски внедрения ИИ | Классификация рисков, стратегии реагирования | LC-7.3 | |

Примеры формулировок приведены в [КИМ-2.4](../../M2-planning-economics/kim-04-test.md). Полные варианты не публикуются в открытой части репозитория и хранятся у преподавателя.
