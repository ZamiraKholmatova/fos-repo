# Модуль 1. Инициация ИИ-проекта и требования к системе ИИ

Модуль посвящен началу жизненного цикла системы ИИ: анализу бизнес-проблемы, оценке перспективности применения ИИ, работе со стейкхолдерами, разработке устава проекта и формированию требований к системе ИИ, включая техническое задание.

**Объем:** 26 а.ч. (лекции — 6, практические занятия — 8, самостоятельная работа — 12).

**Формируемые индикаторы:** LC-1.1, LC-1.3, LC-6.3.

**Логика модуля.** Три КИМ выстроены как последовательное восхождение по ступеням освоения: сначала обучающийся выполняет анализ по образцу (Б₁ → Б₂), затем оформляет результат в проектных документах (Б₂), затем рецензирует чужую работу — действие переноса, требующее применения критериев к незнакомому материалу (**Б₃**). Подробнее — в [траектории освоения](../docs/levels-trajectory.md).
