# Модуль 2. Планирование, риски и экономика ИИ-проекта

Модуль охватывает управление жизненным циклом ИИ-продукта: выбор методологии организации разработки, планирование работ и ресурсов, управление рисками, технико-экономическую оценку, организацию команды и защиту решения перед стейкхолдерами.

**Объем:** 56 а.ч. (лекции — 8, практические занятия — 24, самостоятельная работа — 24).

**Формируемые индикаторы:** LC-4.1, LC-4.2, LC-7.1, LC-7.3.

**Логика модуля.** Обучающийся последовательно строит план проекта (Б₁ → Б₂), рассчитывает экономику и риски (Б₂), организует команду (Б₂), закрепляет теорию на контрольной работе, а затем защищает план на деловой игре в условиях меняющихся ограничений — действие переноса (**Б₃**).
