## 2. Матрица RACI

Обозначения: **R** — исполняет, **A** — отвечает за результат (ровно одна на работу), **C** — с кем консультируются, **I** — кого информируют.

| Работа (уровень 2 ИСР) | Руководитель | Аналитик | Инженер данных | Разработчик | MLOps | Заказчик |
|---|---|---|---|---|---|---|
| | | | | | | |

**Самопроверка:** в каждой строке ровно одна «A»; нет работ без «R»; ни у одного участника не более трех «A» подряд по смежным работам.
